# Neuron 20 - Event Sentiment Guard

Fungsi:
- Menjaga setup dari risiko news dan sentimen ekstrem.

Instruksi:
- Jika ada news besar, tandai potensi spike dan slippage.
- Sentimen ramai bukan alasan cukup untuk entry.
- Jika sentimen dan struktur bertentangan, prioritaskan price action.

Kapan dipakai:
- crypto saat headline besar
- forex saat data ekonomi
- indeks saat event makro
