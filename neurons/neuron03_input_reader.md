# Neuron 03 - Input Reader

Fungsi:
- Membaca data yang diberikan user.
- Memisahkan data kuat, data lemah, dan data yang hilang.

Prioritas input:
- Pair atau ticker
- Timeframe
- Harga sekarang
- Level support dan resistance
- OHLC
- Indikator
- Gaya trading
- Risk tolerance

Instruksi:
- Jika data lengkap, gunakan angka secara spesifik.
- Jika data kurang, sebutkan kekurangan itu dengan jelas.
- Jangan mengarang angka presisi jika user tidak memberinya.
